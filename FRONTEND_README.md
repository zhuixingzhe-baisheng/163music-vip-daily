# 网易云音乐自动任务 - 可视化 Web 管理界面

## 🎉 新增功能

现在项目支持可视化 Web 管理界面了！您可以通过浏览器轻松配置和管理自动任务。

## 快速启动

```bash
# 安装并发运行工具（如果还没有）
npm install -g concurrently

# 同时启动前端和 API 服务
npm run dev:all
```

访问：http://localhost:3000

## 功能特性

### 1. 多账号管理
- 添加多个网易云音乐账号
- 一键切换账号
- 配置导入导出

### 2. 任务配置
- 可视化切换任务开关
- 实时保存配置
- VIP 高级参数配置

### 3. 执行日志
- 查看历史执行记录
- 任务执行详情
- 导出日志文件

## 页面预览

- **首页** - 项目概览和快捷操作
- **账号配置** - 管理网易云音乐账号
- **任务管理** - 配置自动任务参数
- **执行日志** - 查看执行历史

## 技术栈

- Vue 3
- Vite 5
- Pinia（状态管理）
- Vue Router

## 注意事项

- 前端配置保存在浏览器本地存储
- 需要 Node.js 18+ 版本
- 首次使用前请先安装依赖：`npm install`

详细使用说明请参考 [frontend/README.md](./frontend/README.md)
