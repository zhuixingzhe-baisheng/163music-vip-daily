### VIP 音乐任务详解

**4 步流程**:

1. **收藏歌曲** (第 232-244 行)
   - 从 API 动态获取 3 首歌曲
   - 调用 `song_like` API 收藏

2. **上传听歌记录** (第 246-265 行)
   - 使用实际歌曲时长 (`dt` 字段)
   - 调用 `scrobble` API 上报

3. **领取成长值** (第 267-280 行)
   - 检测"收藏三首歌曲"任务
   - 调用 `vip_growthpoint_get` API 领取 (+3 成长值)

4. **取消收藏** (第 282-296 行)
   - 调用 `song_like` API 取消收藏

**使用的 API**:
- `playlist_detail` - 获取歌单详情
- `song_like` - 收藏/取消收藏
- `scrobble` - 上传听歌记录
- `vip_tasks` - 获取 VIP 任务列表
- `vip_growthpoint_get` - 领取成长值

## 🔧 配置说明

### 快速配置

编辑 `auto_tasks_enhanced.js` 第 27 行:

```javascript
cookie: 'MUSIC_U=你的 Cookie 值'
```

### 启用 VIP 音乐任务

编辑 `auto_tasks_enhanced.js` 第 39 行:

```javascript
enableVipMusicTasks: true  // 改为 true 启用
```

### 自定义歌单

编辑 `auto_tasks_enhanced.js` 第 40 行:

```javascript
vipMusicPlaylistId: 8402996200  // 替换为其他歌单 ID
```

## 📝 技术细节

### API 接口

| API | 接口路径 | 方法 | 参数 |
|-----|----------|------|------|
| `vip_tasks` | `/api/vipnewcenter/app/level/task/list` | weapi | - |
| `vip_growthpoint_get` | `/api/vipnewcenter/app/level/task/reward/get` | weapi | `taskIds` |
| `song_like` | `/api/radio/like` | weapi | `id`, `like` |
| `scrobble` | `/api/feedback/weblog` | weapi | `logs` (JSON) |
| `playlist_detail` | `/api/v6/playlist/detail` | weapi | `id` |

### 加密方式

- **加密类型**: weapi (AES-128-ECB + RSA)
- **请求格式**: application/x-www-form-urlencoded
- **响应格式**: JSON (可能带 Brotli 压缩)

### 请求频率

脚本已内置延时以避免风控：
- 收藏/取消：1 秒/首
- 听歌记录：1.5 秒/首
- 多用户：5 秒/用户

### 常见问题
