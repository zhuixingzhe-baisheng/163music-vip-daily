# 网易云 VIP 乐签打卡使用说明

## 功能介绍

本项目支持网易云音乐黑胶 VIP 会员的乐签打卡功能，包括：

### 1. 乐签打卡 (VipSign)
- 每日签到获得成长值 +10
- 自动检测会员状态
- 重复签到自动跳过
- 详细的日志反馈

### 2. 一键领取成长值 (AutoGetVipGrowthpoint)
- 自动领取所有已完成任务的成长值
- 支持听歌、收藏等多个 VIP 任务
- 智能判断可领取任务
- 统计并显示领取的成长值总数

## 配置说明

编辑 `config.json` 文件，设置以下配置项：

```json
{
  "VipSign": true,
  "AutoGetVipGrowthpoint": true
}
```

### 配置项详解

| 配置项 | 类型 | 说明 | 推荐值 |
|--------|------|------|--------|
| `VipSign` | boolean | 是否开启乐签打卡 | `true` |
| `AutoGetVipGrowthpoint` | boolean | 是否自动领取所有 VIP 任务成长值 | `true` |
| `VipListen` | boolean | 是否开启听歌任务 | `false` |
| `VipPlaylistID` | int | 听歌任务使用的歌单 ID | `8402996200` |
| `VipSongLike` | boolean | 是否开启收藏歌曲任务 | `false` |
| `VipSongLikeIDs` | array | 收藏歌曲 ID 列表 | `[]` |
| `ShowVipTaskList` | boolean | 是否显示 VIP 任务列表（调试用） | `false` |

### 推荐配置

```json
{
  "VipSign": true,
  "AutoGetVipGrowthpoint": true,
  "VipListen": true,
  "VipPlaylistID": 8402996200,
  "VipSongLike": true,
  "VipSongLikeIDs": [123456, 789012]
}
```

## 执行流程

完整的乐签打卡流程如下：

1. **检查会员状态** - 验证是否为黑胶 VIP 会员
2. **乐签签到** - 执行每日签到（成长值 +10）
3. **领取签到奖励** - 自动领取签到成长值
4. **检查其他任务** - 查看听歌、收藏等任务进度
5. **一键领取** - 领取所有已完成任务的成长值
6. **结果推送** - 通过 PushPlus/Server酱推送结果

## 成长值获取途径

### 每日必做
- **乐签打卡**: +10 成长值/天
- **听歌任务**: 完成指定时长听歌
- **收藏歌曲**: 收藏指定数量歌曲

### 成长值用途
- 提升黑胶会员等级
- 解锁更多会员权益
- 获得专属标识和装饰

## 常见问题

### Q: 提示"非黑胶会员，跳过乐签打卡"
A: 请确认您的账号已开通黑胶 VIP 会员。

### Q: 乐签打卡失败，代码：-460
A: 表示今日已完成签到，请勿重复签到。

### Q: 领取成长值失败
A: 可能原因：
- 网络连接问题
- Cookie 已过期，请重新获取 MUSIC_U
- 任务尚未完成，请先完成对应任务

### Q: 如何获取 MUSIC_U？
A: 使用项目提供的 QuickLogin 工具，通过网易云客户端扫描二维码登录获取。

## 注意事项

1. **防止风控**: 建议启用 Cron 随机延时功能
2. **Cookie 有效期**: MUSIC_U 有有效期，请定期更新
3. **任务刷新**: VIP 任务每日刷新，请及时完成并领取
4. **账号安全**: 请妥善保管 MUSIC_U，避免泄露
